🚀 CruizControl — Smart Todo App with AI Copilot

CruizControl is a modern productivity web app built with HTML, CSS, and JavaScript, designed to help users manage tasks efficiently with the assistance of an intelligent AI chatbot.

Built by Odumneka Jerry (JerrieCruiz) — JCRUIZ & Co

⸻

✨ Features

📝 Task Management
	•	Add new tasks with priority (High, Medium, Low)
	•	Mark tasks as completed
	•	Delete tasks
	•	Real-time task updates
	•	Search and filter tasks
	•	Dashboard overview (Total, Completed, Pending, High Priority)

⸻

⚡ Dashboard
	•	Clean productivity overview
	•	Task statistics with progress ring
	•	Recent tasks preview
	•	Quick add task section
	•	Daily productivity tips

⸻

🤖 AI Copilot (CruizBot)
	•	Smart chatbot assistant
	•	Refines vague tasks (e.g. “study” → actionable task)
	•	Breaks large tasks into steps
	•	Suggests priorities (high/medium/low)
	•	Provides productivity tips and reminders
	•	Can directly add tasks from chat

⸻

🎨 UI / UX
	•	Modern dark theme design
	•	Smooth animations and transitions
	•	Fully responsive layout
	•	Sidebar navigation system
	•	Clean typography and spacing

⸻

🧠 AI Logic

The AI Copilot (CruizBot) is built using rule-based JavaScript logic, simulating intelligent behavior:
	•	Task refinement engine
	•	Priority detection system
	•	Task breakdown generator
	•	Smart response handling

⚠️ Note: This is a simulated AI system (no external API used)

🛠️ Tech Stack
